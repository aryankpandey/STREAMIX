> A modern Netflix-inspired streaming platform built with cutting-edge web technologies, featuring authentication, movie browsing, trailers, personalized recommendations, and a premium user experience.

## 🚀 Overview

STREAMIX is a full-stack video streaming platform inspired by Netflix. It provides users with a seamless entertainment experience through an intuitive interface, secure authentication, dynamic content discovery, and responsive design.

The project demonstrates modern software engineering practices, scalable architecture, API integration, and production-ready frontend development.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/aryankpandey/streamix.git
cd streamix
```

### Install Dependencies

```bash
# Frontend
cd client
npm install

# Backend
cd ../server
npm install
```

### Environment Variables

Create a `.env` file inside the server directory:

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

TMDB_API_KEY=your_tmdb_api_key
```

### Run Development Server

```bash
# Frontend
npm run dev

# Backend
npm run start
```
